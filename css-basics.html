Что такое CSS?

CSS (Cascading Style Sheets) — каскадные таблицы стилей. Они отвечают за внешний вид веб-страницы: цвета, шрифты, расположение блоков, анимацию. HTML задаёт структуру, а CSS — оформление.

CSS появился в 1996 году и позволил отделить содержимое от представления.

Синтаксис CSS

селектор {
    свойство: значение;
    другое-свойство: значение;
}

Пример:

p {
    color: blue;
    font-size: 16px;
}

Этот код сделает все абзацы синими с размером шрифта 16 пикселей.

Селекторы CSS

  • По тегу: p { } — все абзацы.
  • По классу: .card { } — все элементы с class="card".
  • По id: #header { } — элемент с id="header".
  • Универсальный: * { } — все элементы.

В этом сайте все карточки оформлены через класс .card, а шапка — через тег header.
