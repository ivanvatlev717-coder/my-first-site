Что такое HTML?

HTML (HyperText Markup Language) — язык гипертекстовой разметки. Он описывает структуру веб-страницы с помощью тегов. Браузер читает HTML и отображает содержимое: заголовки, абзацы, картинки, ссылки.

HTML не является языком программирования — это язык разметки. Он был создан Тимом Бернерсом-Ли в 1991 году.

Основные теги (таблица)

Тег Назначение Пример
<h1>...<h6>Заголовки разных уровней<h1>Привет</h1>
<p>Абзац текста<p>Текст...</p>
<a>Гиперссылка<a href="url">Текст</a>
<img>Изображение<img src="photo.jpg" alt="описание">
<ul>, <li>Маркированный список<ul><li>Пункт</li></ul>
<ol>, <li>Нумерованный список<ol><li>Первый</li></ol>
<div>Блочный контейнер<div class="card">...</div>
<span>Строчный контейнер<span style="color:red">важно</span>
<strong>Жирный текст (логическое выделение)<strong>Важно</strong>
<em>Курсив (логическое выделение)<em>курсив</em>
<br>Перенос строкиТекст<br>новая строка
<hr>Горизонтальная линия<hr>

Важно: каждый открывающий тег должен иметь закрывающий (кроме самозакрывающихся, например <img>, <br>).

Пример простой HTML-страницы

<!DOCTYPE html>
<html>
<head>
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый сайт.</p>
    <a href="https://example.com">Ссылка</a>
</body>
</html>

Сохраните этот код в файл index.html и откройте в браузере — вы увидите свою первую веб-страницу!
