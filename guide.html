Веб-старт

Главная HTML CSS Гайд О проекте

Создайте свой первый сайт за 15 минут!

Следуйте этим пяти шагам — и у вас будет работающая веб-страница. Не нужны специальные программы, только блокнот и браузер.

Шаг 1. Создайте папку и файл

Создайте на рабочем столе новую папку, назовите её мой_сайт. Внутри папки создайте текстовый документ и переименуйте его в index.html (расширение должно быть .html, а не .txt).

Совет: включите в проводнике отображение расширений файлов.

Шаг 2. Напишите базовую структуру HTML

Откройте файл index.html в любом текстовом редакторе (Блокнот, VS Code, Notepad++). Скопируйте и вставьте следующий код:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый сайт, созданный с нуля.</p>
</body>
</html>

Шаг 3. Сохраните файл

Нажмите Ctrl+S (или Cmd+S на Mac), чтобы сохранить изменения. Убедитесь, что файл имеет расширение .html.

Шаг 4. Откройте в браузере

Найдите файл index.html в папке, кликните по нему правой кнопкой мыши и выберите «Открыть с помощью» → ваш браузер (Chrome, Firefox, Edge). Или просто дважды кликните по файлу — он откроется в браузере по умолчанию.

Вы увидите заголовок «Привет, мир!» и абзац. Поздравляю, вы создали сайт!

Шаг 5. Внесите изменения

Вернитесь в текстовый редактор, измените текст в тегах <h1> и <p>. Сохраните файл и обновите страницу в браузере (клавиша F5). Изменения появятся сразу!

Попробуйте добавить:

  • Список ваших любимых книг (теги <ul> и <li>)
  • Ссылку на любой сайт (<a href="https://example.com">Текст</a>)
  • Изображение (скачайте картинку и используйте <img src="название.jpg">)
